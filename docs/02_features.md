# Features

1. Fetch random high-quality images from the Unsplash API.
2. Display the fetched images in a scrollable list or grid (RecyclerView).
3. Provide a refresh action (like a button or swipe-to-refresh) to load a new set of images.
4. Show a loading indicator while the images are being fetched from the internet.
5. Display basic information for each image (e.g., author name or a short description).
