# AI Agent Guidelines
The AI agent must follow a planning-first approach.

Rules:
1. Always read the documentation inside /docs before generating code.
2. Follow the architecture defined in docs/06_architecture.md.
3. Generate Kotlin code only.
4. UI must use XML Views (Do not use Jetpack Compose).
5. Do not generate large files at once.
6. Generate code step-by-step following the implementation plan.
